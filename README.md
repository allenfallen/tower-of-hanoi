# Tower of Hanoi
An iterative C++ solution to the Tower of Hanoi puzzle using SFML to animate and display the game.

# Keys
 - `space` to trigger a move
 - `q` to return to title screen and quit

# Dependencies
- [SFML 2.3.2] (https://www.sfml-dev.org/)
